Snapshot

A commit captures the state of every tracked file at a point in time.

Message

Every commit has a human-readable message describing what changed and why.

Linked

Each commit points to its parent, forming a chain you can walk through.

Permanent

Once committed, changes are safely stored. You can always return to any commit.
