Git Workflow Diagrams

Interactive animated tutorials for standard Git operations

1. Git Clone
Understand how retrieving a remote repository into a local workspace operates.
2. Git Branch & Commits
Visualize isolating features in branches and structurally adding new commits.
3. Git Push
Learn how to upload local branch commits back up to the remotes.
4. Merge Requests
Walk through reviewing code and merging Pull Requests via the remote.
5. Merge Conflict
Simulate a failed git pull caused by conflicting Python edits on the same file, then resolve it step by step.
6. Git Pull
Sync a local workstation repository with the newly merged remote.
7. Diverged History
Visualize the common error problem of attempting to push when tracking branches have unsynced commits.
8. Git Rollback
Understand how to drop broken commits and completely undo mistakes.
9. Git Squash
Clean up your local commit history by squashing multiple commits into one.
10. Git Cherry-Pick
Grab an exact single commit from another branch and surgically apply it to yours.
11. Git Merge
Combine multiple divergent branch histories together using a new merge commit.
12. Git Rebase
Rewrite history by moving your branch to structurally start from the tip of another.
13. Git Stash
Learn how to securely pocket unfinished code changes so you can temporarily context switch.
