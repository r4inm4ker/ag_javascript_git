The Big Picture

In a professional software studio, writing code is only one piece of the puzzle. You also need to track changes, build packages, and manage dependencies. Three specialised tools work together to cover this entire lifecycle: Git, Waf, and Oz.

Git

Version Control — track every change to your source code over time.

Waf

Build System — compile and package your code into versioned releases.

Oz

Environment Manager — resolve dependencies and configure your shell.
