Welcome

Let's Git Gud is an interactive, visual tutorial series designed to teach you Git from the ground up. Each lesson walks you through a core concept with step-by-step animations — no prior experience required.

You'll learn how code flows through a modern development pipeline: from your very first commit, through branching and merging, all the way to collaborating with a team via remote repositories.
